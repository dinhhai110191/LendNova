# LendNova

LendNova is a decentralized lending and borrowing
protocol built for scalable DeFi money markets.

The protocol enables users to:

- Supply crypto assets
- Borrow against collateral
- Earn passive yield
- Participate in governance
- Access cross-chain liquidity

---

# Vision

LendNova aims to create a secure,
capital-efficient decentralized
money market ecosystem.

The protocol focuses on:

- Real yield generation
- Secure lending markets
- AI-powered risk management
- Multi-chain scalability
- DAO governance
